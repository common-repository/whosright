== Description ==

Whosright polling and commenting system powered by http://whosright.com/ API. You need to sign up on http://whosright.com 
and get your API key under http://whosright.com/settings in order to enable the plugin functionality.

== Installation ==

After activating the plugin, go to Settings -> Whosright and input your Whosright API key in order to start using the plugin. 
You can get a Whosright API key by signing up on http://whosright.com/ and going to Settings -> Add API Key
